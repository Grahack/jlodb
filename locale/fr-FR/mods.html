Mods

Un mod (pour module) est une application qui fait usage d'au moins 2 activités différentes issues de la base de données jLoDB. Celui-ci peut disposer également d'une base de données propre qu'il est nécessaire d'installer préalablement.

La liste des mods installés sur ce serveur est fournie ci-dessous. Cliquez sur l'icône de l'application pour lancer cette dernière.
